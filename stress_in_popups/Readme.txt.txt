Tested with Firefox version: 52.09.0 on Windows XP

How to install 
1)  Place a copy of "dict.fsa" in the same folder as "single_word_stress_server.rb"
2)  Launch "single_word_stress_server.rb"
3) Type "about:debugging" in the address bar of Firefox.
4) Click the button "Load  temporary add-on"
5) Navigate to the directory  "webXtension" and select one file

How to use
1) Open a russian web page
2) Select a single  word ( small cyrillic characters only!)
Then a tooltip pops up.  Words  recognized by the automaton"dict.fsa"  appear fully stressed in the tooltip. A single click closes the tooltip.
