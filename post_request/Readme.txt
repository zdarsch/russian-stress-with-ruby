Tested with Ruby 1.8.7 and  Firefox version: 52.09.0 on Windows XP

How to use
1)  Place a copy of "dict.fsa" in the same folder as "post_stress_server.rb"
2)  Launch "post_stress_server.rb"
3) Type "about:debugging" in the address bar of Firefox.
4) Click the button "Load  temporary add-on"
5) Navigate to the directory  "post_request" and select one of the files
6) Open a russian web page in Firefox.
